Instruções

Esta aplicação serve para registar as ações efetuadas pelo jogador que observará, já que esses eventos registados serão armazenados numa base de dados, para que sejam consultados no futuro por qualquer pessoa.

Botão "Interrupção" - este botão é para ser clicado quando o jogo estiver parado, para que o utilizador tenha tempo de registar o evento efetuado pelo jogador que está a observar.

Botão "Perda de Bola" - este botão é para ser clicado quando o jogador que está a observar perder a bola.

Botão "Recuperação de Bola" - este botão é para ser clicado quando o jogador que está a observar recuperar a bola.

Botão "Equipa Perdeu a Bola" - este botão é para ser clicado quando a equipa do jogador que está a observar perder a bola.

Botão "Equipa Recuperou a Bola" - este botão é para ser clicado quando a equipa do jogador que está a observar recuperar a bola.
